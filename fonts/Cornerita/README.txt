Cornerita Variable Font
=======================

This download contains Cornerita as both a variable font and static fonts.

Cornerita is a variable font with these axes:
  wght

This means all the styles are contained in a single file:
  Cornerita-VF.ttf

If your app fully supports variable fonts, you can now pick intermediate styles
that aren’t available as static fonts. Not all apps support variable fonts, and
in those cases you can use the static font files for Cornerita:
  static/Cornerita Thin.ttf
  static/Cornerita Regular.ttf
  static/Cornerita Bold.ttf
  static/Cornerita Black.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
